Always write Progress Notes entries in descending reverse-chronological order (newest entry at the top).

**Why:** User corrected chronological ordering on 05-hearts after the fact — reverse-chron is the expected default.

**How to apply:** Any time Progress Notes are added to or written for a card file, place new entries at the top of the list, not the bottom. If writing a full set of notes from scratch, order them newest-first.
