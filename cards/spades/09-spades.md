## Goal Description

Read Computer Systems: A Programmer's Perspective by Bryant and O'Hallaron. Bridges the gap between high-level programming and the underlying hardware and OS. Covers how programs are compiled, executed, and interact with the operating system. Brings together computer architecture and OS knowledge in a programmer-focused context.
